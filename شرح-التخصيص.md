# شرح تخصيص موقع المطعم

هذا الموقع معمول كقالب جاهز لمطاعم الوجبات السريعة. كل النصوص والأسعار والصور موجودة في ملف واحد:

`data/restaurant.json`

## طريقة تغيير بيانات المطعم

افتح ملف `data/restaurant.json` وعدل هذه القيم:

- `name`: اسم المطعم
- `slogan`: الجملة التسويقية الرئيسية
- `phone`: رقم الهاتف للاتصال المباشر
- `whatsapp`: رقم واتساب بصيغة دولية، مثل `+213555000000`
- `address`: عنوان المطعم
- `hours`: أوقات العمل
- `mapQuery`: النص الذي سيبحث عنه Google Maps
- `heroImage`: صورة كبيرة وجذابة تظهر في أول الموقع

## تعديل المنيو

كل منتج داخل `menu` يحتوي على:

- `category`: التصنيف مثل Burgers أو Pizza
- `name`: اسم المنتج
- `description`: وصف قصير يفتح الشهية
- `price`: السعر
- `popular`: ضع `true` لإظهار شارة Popular
- `image`: رابط صورة المنتج

مثال:

```json
{
  "category": "Burgers",
  "name": "Double Flame Burger",
  "description": "Two smashed beef patties, cheddar, caramelized onions, signature flame sauce.",
  "price": 9.99,
  "popular": true,
  "image": "https://images.unsplash.com/photo-1568901346375-23c9450c58cd?auto=format&fit=crop&w=900&q=80"
}
```

## تشغيل الموقع

من داخل مجلد المشروع شغل:

```bash
npm start
```

ثم افتح:

```text
http://127.0.0.1:4173
```

## أهم ميزات التحويل

- زر طلب ثابت في أسفل الشاشة
- زر واتساب عائم
- زر اتصال مباشر
- سلة بسيطة ترسل الطلب مباشرة على واتساب
- عروض محدودة الوقت لخلق الاستعجال
- تقييمات عملاء لبناء الثقة
- تصميم Mobile-first مناسب لمعظم زوار المطاعم

## نصائح لجعل الموقع يبيع أكثر

- استخدم صور حقيقية عالية الجودة لأفضل المنتجات.
- اجعل أول عرض واضحًا وقويًا، مثل Combo أو خصم اليوم.
- ضع رقم واتساب صحيحًا لأن الطلب يفتح برسالة جاهزة.
- اجعل الوصف قصيرًا وشهيًا، ولا تكثر الكلام داخل كروت المنتجات.
- حدث الأسعار والعروض بشكل دوري.
